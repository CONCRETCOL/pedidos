Configura Firebase para sincronizar en tiempo real

  1. Ve a console.firebase.google.com y crea un proyecto gratuito
  2. En el proyecto, ve a Firestore Database → Crear base de datos → Modo producción
  3. Ve a Configuración del proyecto → Agrega una app web → Copia los datos
  4. Pega los datos aquí abajo y haz clic en Guardar configuración

También debes configurar las Reglas de Firestore así:
allow read, write: if true; (por ahora, luego puedes agregar autenticación)
